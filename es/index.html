¿Qué es LitecoinZ?

LitecoinZ es una criptomoneda descentralizada basada en bitcoin para ser compatible con el núcleo de ZCash. Utiliza el algoritmo Equihash con zk-SNARK, direcciones T y direcciones Z anónimas. Al combinar las tecnologías, LitecoinZ puede operar como una criptomoneda con la capacidad de proporcionar una tarifa baja y una transacción rápida a personas de todo el mundo.

LitecoinZ es el primer proyecto desarrollado 100% de la comunidad para la comunidad y el primer fork de z-coin con una GUI nativa para Linux / Windows / Mac OS.

Recompensa de Minado

Los mineros actualmente reciben 50 nuevos LitecoinZ por bloque, una cantidad que se reduce a la mitad cada 4 años (cada 840,000 bloques). Por lo tanto, la red LitecoinZ está programada para producir 84 millones de litecoinZ, que es 4 veces más grande que Bitcoin.
