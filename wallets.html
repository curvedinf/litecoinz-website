Wallets

Full Node — LitecoinZ Core

LitecoinZ Core is the reference node and wallet for LitecoinZ. It includes all features of the network and is based on Bitcoin Core, the original Bitcoin software. Being a full node, it must download the entire LitecoinZ blockchain before being able to make new transactions.

Web Wallet

The LitecoinZ web wallet is a quick way to create or access a wallet. The web wallet lacks many features of the full node, but does offer availability for any device with a browser and does not need to download the blockchain before use.

3rd Party Multi-coin Wallet — Hebe Wallet

Hebe Wallet is a mobile and browser-based wallet that supports LitecoinZ and many other cryptocurrencies and tokens.
