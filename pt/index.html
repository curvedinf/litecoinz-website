O que é o LitecoinZ ?

O LitecoinZ é um bitcoin descentralizado baseado em criptomoedas compatível com o núcleo Zcash. Ele utiliza o algoritmo Equihash com snarks Z, endereços T (públicos) e endereços Z anônimos. Combinando as tecnologias, a LitecoinZ é capaz de operar como criptomoeda, com a capacidade de fornecer mineração de unidades de processamento para pessoas em todo o mundo.

O LitecoinZ é o primeiro projeto 100% desenvolvido a partir da comunidade para a comunidade e o primeiro fork da z-coin com uma GUI (Carteira) nativa para Linux / Windows / Mac OS.

Recompensa de Mineração

Mineradores são atualmente recompensados com 50 LitecoinZ por bloco, essa recompensa que é diminuida pela metade a cada 4 anos (a cada 840.000 blocos). A rede LitecoinZ está, portanto, programada para produzir 84 milhões de litecoinZ, que é 4 vezes maior que Bitcoin.
